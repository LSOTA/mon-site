Chiffrement d’appareil dans Windows

Qu'est-ce que le chiffrement d’appareil ?

Le chiffrement d’appareil est une fonctionnalité Windows qui active automatiquement le chiffrement BitLocker pour le lecteur du système d’exploitation et les lecteurs fixes. Il est particulièrement avantageux pour les utilisateurs quotidiens qui souhaitent s’assurer que leurs informations personnelles sont sécurisées sans avoir à gérer des paramètres de sécurité complexes.

Activation du chiffrement d’appareil

  • Se connecter à Windows avec un compte d’administrateur.
  • Dans l’application Paramètres sur votre appareil Windows, sélectionnez Confidentialité > Chiffrement de l’appareil.
  • Utilisez le bouton bascule pour activer le chiffrement de l’appareil.

Pourquoi le chiffrement d’appareil n’est-il pas disponible ?

Voici les étapes pour déterminer pourquoi le chiffrement d’appareil n’est peut-être pas disponible :

  • Dans Démarrer, tapez Informations système, cliquez avec le bouton droit sur Informations système dans la liste des résultats puis sélectionnez Exécuter en tant qu’administrateur.
  • Dans la liste Résumé du système, recherchez la valeur prise en charge du chiffrement automatique des appareils.
  • La valeur fournit les raisons pour lesquelles le chiffrement d’appareil ne peut pas être activé.

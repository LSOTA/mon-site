Chiffrement de lecteur BitLocker

Introduction

Le chiffrement de lecteur BitLocker vous permet de chiffrer manuellement un ou plusieurs lecteurs spécifiques sur un appareil exécutant Windows Pro, Entreprise ou Éducation. Pour les appareils gérés par une organisation, le chiffrement de lecteur BitLocker est généralement géré par le service informatique, car le chiffrement peut être requis par les stratégies de l’organisation.

Accéder au chiffrement de lecteur BitLocker

Voici les étapes pour accéder au chiffrement de lecteur BitLocker :

  • Se connecter à Windows avec un compte d’administrateur.
  • Dans Démarrer, tapez BitLocker et sélectionnez Gérer BitLocker dans la liste des résultats.
  • Remarque : Le chiffrement de lecteur BitLocker est disponible uniquement sur les appareils exécutant Windows Pro, Entreprise ou Éducation.

Chiffrer manuellement un lecteur

Pour chiffrer manuellement un lecteur :

  • Ouvrir le chiffrement de lecteur BitLocker.
  • À côté de chaque lecteur, sélectionnez l’option Activer BitLocker.
  • Sélectionner une option de déverrouillage et sauvegarder la clé de récupération.
  • Le lecteur commence le processus de chiffrement. Cela peut prendre un certain temps, mais vous pouvez continuer à utiliser votre appareil pendant qu’il progresse.
